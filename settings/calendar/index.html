How to find your ICS URL (all major providers)
  • · Google Calendar: Settings → pick your calendar on the left → "Integrate calendar" → Secret address in iCal format.
  • · Outlook.com: Settings → View all Outlook settings → Calendar → Shared calendars → "Publish a calendar" → choose permission "Can view all details" → copy the ICS link.
  • · Office 365: Same path; if admin-locked, ask IT to enable calendar publishing or use the OAuth connect above.
  • · iCloud: iCloud.com → Calendars → ⓘ next to the calendar → "Public Calendar" → copy link (replace webcal:// with https://).
  • · Fastmail: Calendars → ⚙ → Publish → copy ICS URL.
